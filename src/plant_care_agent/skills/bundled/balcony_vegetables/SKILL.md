# 阳台蔬菜规划 Skill

## 1. 环境与约束
- 确认朝向、每日直射光时长、是否有封闭玻璃、冬季最低温。
- 用 `weather_forecast` 拉取当地未来 7 天温湿度与降雨，提示霜冻/连续阴雨风险。

## 2. 选种与搭配
- 对用户目标作物调用 `plant_knowledge`，核对：光照需求、容器最小体积、间距、伴生/忌种。
- 给出 **3 套** 可选组合（例如：速生叶菜 / 番茄+罗勒 / 辣椒+葱），说明难度与采收节奏。

## 3. 两周执行表
- 调用 `care_scheduler`，把用户环境摘要写进 `plant_info` 参数（含地区、光照、容器类型）。
- 将日程中的浇水/施肥频率与天气挂钩说明（如高温大风日增加叶面补水）。

## 4. 风险与预案
- 阳台特有风险：徒长、红蜘蛛、霉病；各给 1 条预防 + 1 条早期处理。
- 建议用户用 `growth_journal` 固定记录「品种 | 事件」，便于多盆管理。

## 5. 交付物
- 输出：**推荐品种清单**、**采购/播种时间表**、**首月检查清单**（每周 1 次）。
