# 轮作规划指南

## 一、基础原则

### 1. 科属轮换
同科作物不宜连续种植，建议至少间隔 2-3 季：
- 茄科（番茄、辣椒、茄子、马铃薯）
- 葫芦科（黄瓜、西瓜、南瓜）
- 十字花科（白菜、萝卜、花菜）
- 豆科（大豆、绿豆、花生）— 固氮作物
- 禾本科（水稻、小麦、玉米）

### 2. 营养互补
- 重氮作物（叶菜类）→ 豆科（固氮）→ 重磷钾作物（根茎/果实类）
- 深根 → 浅根交替

### 3. 病虫害轮换
- 不同科作物的病虫害谱不同
- 禾本科可打断多数蔬菜病害循环
- 豆科 + 深翻可降低线虫密度

## 二、规划流程

### 1. 数据收集
- 调用 `list_farm_zones` 查看各地块当前作物
- 调用 `query_farm_history` 查看历史种植记录
- 调用 `read_sensors` 获取土壤 pH 和湿度现状
- 调用 `weather_forecast` 了解未来季节气候趋势

### 2. 土壤评估
| 土壤pH | 适宜作物 |
|--------|----------|
| 5.0-5.5 | 蓝莓、茶叶 |
| 5.5-6.5 | 马铃薯、西瓜、花生 |
| 6.0-7.0 | 大部分蔬菜、水稻 |
| 7.0-7.5 | 大豆、玉米、小麦 |
| 7.5-8.0 | 棉花、甜菜 |

### 3. 经典轮作模板

**水田轮作（华东/华南）：**
春：水稻 → 夏：水稻（双季稻）→ 冬：绿肥（紫云英）

**旱地轮作（华北）：**
春：玉米 → 秋冬：小麦 → 春：大豆 → 秋冬：小麦

**蔬菜轮作（大棚）：**
春：番茄 → 夏：黄瓜 → 秋：白菜 → 冬：菠菜

## 三、输出方案

使用 `log_farm_event` 记录轮作规划决策，包含：
- 当前作物和下一季计划
- 轮作理由（营养/病害/市场）
- 预计播种和收获时间
- 需要的土壤改良措施
