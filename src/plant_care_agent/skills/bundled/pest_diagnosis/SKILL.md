# 病虫害排查 Skill

按以下顺序执行，并在回复中分节给出结论与行动项。

## 1. 信息收集
- 确认植物种类、生长阶段、摆放环境（室内/阳台/露养）、近 7 天浇水/施肥/通风变化。
- 若有照片：调用 `plant_image_analyzer` 做视觉诊断，记录其结论作为参考而非唯一依据。

## 2. 知识检索
- 对该植物调用 `plant_knowledge`，重点阅读「常见虫害」「常见病害」条目。
- 若现象仍不明确：用 `web_search` 检索「{植物名} + 症状关键词 + 防治」。

## 3. 鉴别要点（简要自检）
- 虫害：是否有活体虫、蛛网、黏液、不规则啃食。
- 真菌：霉层、同心轮纹、潮湿环境下扩散。
- 生理：均匀黄化多为浇水/光照/缺素；斑点集中在叶尖多为肥害或水质。

## 4. 建议输出格式
1. **最可能原因**（1～3 条，按概率排序）
2. **立即可做**（隔离、修剪病叶、改善通风、停肥等）
3. **用药/生物防治**（低毒优先；说明适用场景与安全间隔）
4. **何时复查**（例如 3～5 天后拍照对比）

## 5. 记录
- 若用户同意，用 `growth_journal` 以 `病害` 或 `虫害` 类型记录本次判断与处理，便于后续追踪。
